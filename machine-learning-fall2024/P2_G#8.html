Notebook comment on ARI¶

Adjusted Rand Index of the KMeans clustering model is 0.028. Adjusted Rand Index of the Agglomerative clustering model is 0.040. The clustering algorithm with a better outcome on the crabs dataset using the ARI measure is the agglomerative clustering model. Although not much higher than the kmeans clustering model, the agglomerative clustering model has an ARI closer to 1. Similar to a correlation coefficient, a value closer to 1 indicates a stronger and intuitive cluster relationship in varying shapes and sizes. The ARI takes the ground truth values and predicted cluster labels as parameters for the ARI function, which indicates how well the predicted cluster labels match the predefined ground truth labels/classifications.
